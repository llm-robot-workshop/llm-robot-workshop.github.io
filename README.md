# Workshop 1 Website

This is a static GitHub Pages website for:

**LLM/Human Augmented Robot Learning and Planning**

## Files

- `index.html`: Home page
- `call-for-contributions.html`: Call for Contributions page
- `organizers.html`: Organizers page
- `style.css`: Website style
- `assets/`: Images and icons

## How to publish with GitHub Pages

1. Create a public GitHub repository, for example `icrl-workshop-1`.
2. Upload all files in this folder to the repository root directory.
3. Go to `Settings` → `Pages`.
4. Choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save and wait for GitHub Pages to generate the website link.
